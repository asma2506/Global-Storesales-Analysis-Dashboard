# Global Superstore Sales Analysis Dashboard

## Project Overview
This project analyzes Global Superstore sales data using Power BI to identify sales trends, profitability, customer behavior, and market performance.

## Tools Used
- Power BI
- Power Query
- DAX

## Dataset
Global Superstore Dataset

## Dashboard Features
- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Sales by Category
- Profit by Sub-Category
- Sales Trend
- Sales by Market
- Sales by Region
- Top 10 Products by Sales
- Top 10 Customers by Sales

## Key Insights
- Technology category generated the highest sales.
- Copiers delivered the highest profit.
- APAC was the top-performing market.
- Sales showed consistent growth over time.
